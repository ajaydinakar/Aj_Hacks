# **installing http server using the npm**
~~~
npm install -g http-server
~~~

# **Starting the http server**
~~~
 http-server -c-1
~~~

# **output will be like below**
~~~
Starting up http-server, serving ./
Available on:
  http://192.168.1.78:8081
  http://127.0.0.1:8081
Hit CTRL-C to stop the server

~~~
